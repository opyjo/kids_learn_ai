# Term 3, Lesson 6: Story Choices! 📖

## Teacher's Guide

**Course:** Term 3: Decision Maker

## 📋 Lesson Overview

### Purpose

This lesson teaches students to build interactive stories using if/else statements. This is a fun, creative application that prepares them for the term project (Mini Adventure Game) next week. Students combine storytelling with programming to create choose-your-own-adventure stories.

### Learning Objectives

By the end of this lesson, students will be able to:

1. Build interactive stories with choices
2. Use if/else to create different story paths
3. Combine input() with if/else for stories
4. Create choose-your-own-adventure stories
5. Prepare for the term project

### Materials Needed

- Computer with internet access (for each student)
- Projector/screen share for demonstrations
- Trinket accounts
- Whiteboard for story planning

### Pre-Lesson Preparation

1. **Review if/else concepts** — Ensure you're comfortable
2. **Prepare story examples** — Have example stories ready
3. **Think of story themes** — Age-appropriate themes
4. **Prepare story planning framework** — Help students plan

## 📚 Detailed Teaching Guide

### Part 1: Review & Introduction (5 minutes)

#### Goals

- Quick review of if/else
- Build excitement about stories
- Introduce interactive stories

#### Script/Talking Points

> "Welcome back! You've learned to make programs that decide. Today, we're going to use that to tell STORIES! Interactive stories that change based on what the reader chooses!"

**Quick Review:**
- "What does if/else do?" → Makes decisions
- "What do we use to get input?" → input()

> "Perfect! Now let's tell stories!"

#### Teaching Tips

- Build excitement about storytelling
- Connect to choose-your-own-adventure books
- Emphasize creativity

---

### Part 2: What Are Interactive Stories? (15 minutes)

#### Goals

- Introduce interactive stories
- Build example together
- Show how if/else creates different paths

#### Live Coding: Build Story Together

**Introduction:**
> "An interactive story is a story where the reader makes choices, and the story changes! Let's build one together!"

**Build together:**
```python
print("=== THE MYSTERIOUS DOOR ===")
print("You're walking through a forest and find a door in a tree.")
print("")

choice = input("Do you want to open it? (yes/no): ")

if choice == "yes":
    print("You open the door and find... a tiny dragon!")
else:
    print("You walk away.")
```

**Test it:**
- Try "yes"
- Try "no"
- Show both paths work!

#### Teaching Tips

- Build together so everyone has a working start
- Emphasize: "Now make it yours!"
- Show how choices create different paths

---

### Part 3: Story Building Workshop (30 minutes)

#### Goals

- Students build their own stories
- Apply if/else to storytelling
- Get help as needed

#### Story Planning Framework

Help students plan:
1. **Beginning** — What's the story about?
2. **First choice** — What choice does the reader make?
3. **If yes** — What happens?
4. **If no** — What happens?
5. **Ending** — How does it end?

#### Teacher Circulation

During this time:
- Help students plan stories
- Guide through building
- Check syntax and structure
- Celebrate creative stories
- Help debug issues

---

### Part 4: Share & Wrap-up (7 minutes)

#### Goals

- Students share stories
- Celebrate creativity
- Introduce homework

#### Sharing

> "Who wants to share their story? Show us what happens with different choices!"

#### Introduce Homework

> "For homework, create an interactive story with at least 3 choices and 2 endings!"

#### Preview Next Week

> "Next week, we'll build your BIG term project: a complete mini adventure game! Get ready!"

## 🎯 Assessment & Differentiation

### Formative Assessment

**During class, observe:**
- [ ] Can student build interactive stories?
- [ ] Does student use if/else correctly?
- [ ] Is student creative with choices?
- [ ] Does student test both paths?

### Homework Assessment

| Criteria                | Exceeds (3)                      | Meets (2)              | Developing (1)        |
| ----------------------- | -------------------------------- | ---------------------- | --------------------- |
| **Number of choices**    | 5+ choices, creative             | 3+ choices             | Fewer than 3          |
| **Number of endings**    | 3+ different endings             | 2+ endings              | Only 1 ending         |
| **Uses if/else**         | All choices use if/else correctly | Most use if/else       | Missing if/else       |
| **Story quality**        | Creative, engaging story         | Clear story             | Unclear story         |
| **Comments**            | Clear, helpful comments          | Some comments          | No comments           |
| **Code runs**           | No errors, works perfectly      | Minor issues, runs     | Major errors          |

### Differentiation Strategies

#### For Students Who Need More Support

- Provide story template
- Start with just 1-2 choices
- Focus on getting it working
- Pair with peer
- Help with planning

#### For Advanced Students

- Challenge with multiple nested choices
- Multiple endings
- Complex story branches
- Help peers
- Create story collections

## ⚠️ Common Pitfalls & Solutions

### Technical Issues

| Problem                     | Prevention                          | Solution                                      |
| --------------------------- | ----------------------------------- | --------------------------------------------- |
| Syntax errors               | Type slowly, check structure        | "Check: if, condition, :, indentation"        |
| Not testing both paths      | Emphasize testing                   | "Test with both choices!"                     |

### Conceptual Issues

| Problem                     | Signs                               | Solution                                      |
| --------------------------- | ----------------------------------- | --------------------------------------------- |
| Doesn't know what to write  | Stuck, not starting                 | "What story do YOU want to tell?"             |
| Can't plan story            | Overwhelmed                         | "Let's plan it together: beginning, choice, ending" |

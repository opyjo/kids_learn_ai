## Class Activity: Coding Olympics! 🏅

### Setup
Divide class into teams of 2-3 students.

### Event 1: Speed Print (5 min)
First team to correctly print this EXACT output wins:

```
Hello, World!
I love Python!
Coding is fun!
```

### Event 2: Variable Relay (8 min)
Each team member adds one variable, then combines them:

- Student 1: Creates `adjective` variable
- Student 2: Creates `animal` variable  
- Student 3: Creates `action` variable
- All together: Print "The [adjective] [animal] [action]!"

### Event 3: Transformation Challenge (7 min)
Given: `message = "python is awesome"`

Transform it to show:
1. All uppercase
2. Title case
3. Repeated 3 times
4. Length

First team with all 4 correct wins!

### Event 4: Creative Sprint (10 min)
Create the most CREATIVE program using:
- At least 3 variables
- At least 5 print statements
- At least 1 string method

Teams vote on most creative (can't vote for own team).

### Event 5: Debug Race (5 min)
Fix this broken code as fast as possible:

```python
Name = "alex"
print("Hello" name)
print(Name.UPPER())
print("Goodbye)
```

### Awards Ceremony
- Gold Medal: Overall winner
- Creativity Award: Most creative program
- Team Spirit Award: Best collaboration

## Take-Home Assignment: Practice Collection 📚

### Instructions

Complete at least 3 of these 5 practice challenges:

**Challenge A: Personal Info Card** ⭐
Create variables for name, age, school, favorite subject. Print them nicely with borders.

**Challenge B: Silly Story** ⭐
Create a mad-libs style story with at least 3 adjectives, 2 nouns, 1 verb, 1 place.

**Challenge C: ASCII Art** ⭐⭐
Create ASCII art of your first initial, an animal, vehicle, or house using at least 5 print statements.

**Challenge D: Word Analysis** ⭐⭐
Pick a favorite word and show it in lowercase, UPPERCASE, Title Case, show its length, and repeat it 5 times.

**Challenge E: Greeting Generator** ⭐⭐⭐
Create a program that makes a greeting card with person's name variable, occasion variable, decorated border, and personalized message.

### Grading
- Complete 3 challenges: Full credit
- Complete all 5: Bonus points!
- Extra creative solutions: Extra bonus!

## Common Student Challenges

| Challenge | Solution |
|-----------|----------|
| Forgetting syntax from earlier weeks | Quick review at start |
| Rushing without testing | Emphasize "test after each line" |
| Getting stuck on one challenge | Move on, come back later |
| Comparing to others | Focus on personal growth |

---

## Differentiation

### For Struggling Students
- Let them choose 2 challenges instead of 3
- Pair with supportive partner
- Provide starter code templates

### For Advanced Students
- Challenge to complete all 5
- Add extra requirements (more variables, more methods)
- Time challenges for extra difficulty

## Class Activity: Message Experiment Lab 🧪

### Purpose
Help students discover print() capabilities through experimentation.

### Part 1: What Will Happen? (5 min)

Before running each code, students PREDICT what will happen:

```python
print("Hello" * 3)
```
Expected: `HelloHelloHello`

```python
print("5 + 5")
```
Expected: `5 + 5` (not 10 - it's text!)

```python
print("")
print("Surprise!")
```
Expected: Blank line, then "Surprise!"

### Part 2: Creative Challenge (15 min)

Create the MOST CREATIVE output using only print()!

**Rules:**
- Use at least 5 print statements
- Use at least one print trick (empty line, multiplication, or both)
- Make it look interesting!

**Ideas:**
- A face made of text
- A simple picture
- A formatted poem
- A mini story

### Part 3: Show and Tell (10 min)

- Share creation with a partner
- Explain what print tricks used
- Give each other one compliment and one suggestion

### Bonus Challenge ⭐
Can you make a box around text like this?
```
+------------+
|  My Name   |
+------------+
```

## Take-Home Assignment: My Print Masterpiece 🎨

### Requirements

Student's program must have:

1. ✅ At least **8 print statements**
2. ✅ At least **one empty line** (blank space)
3. ✅ At least **one repeated character** (using `*`)
4. ✅ A **title** at the top
5. ✅ Be **creative and fun**!

### Grading Rubric

| Criteria | Points |
|----------|--------|
| Has 8+ print statements | 20 |
| Uses empty line | 15 |
| Uses string multiplication | 15 |
| Has a title | 15 |
| Creative and original | 20 |
| Code runs without errors | 15 |
| **Total** | **100** |

---

## Common Student Challenges

| Challenge | Solution |
|-----------|----------|
| Confusing * for math vs text | Demonstrate: `"Hi" * 3` vs `5 * 3` |
| Quote matching errors | Color-code quotes on board |
| Empty lines not working | Make sure it's `print("")` not `print(" ")` |

---

## Differentiation

### For Struggling Students
- Provide templates with blanks to fill in
- Focus on 5 print statements instead of 8
- Pair programming with stronger student

### For Advanced Students
- Challenge to create ASCII art
- Create complex borders
- Combine string multiplication creatively

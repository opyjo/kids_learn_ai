## Printing Multiple Messages

Python reads code from **top to bottom**, like reading a book!

```python
print("Line 1: Hello!")
print("Line 2: How are you?")
print("Line 3: I am Python!")
```

**Output:**
```
Line 1: Hello!
Line 2: How are you?
Line 3: I am Python!
```

Each `print()` command creates a **new line**!

## Print Tricks! ✨

### Trick 1: Empty Lines

Want a blank line? Use empty quotes!

```python
print("Hello!")
print("")
print("Goodbye!")
```

**Output:**
```
Hello!

Goodbye!
```

### Trick 2: Repeating Text

You can multiply text with `*`!

```python
print("Ha" * 3)
print("-" * 20)
print("🎉" * 5)
```

**Output:**
```
HaHaHa
--------------------
🎉🎉🎉🎉🎉
```

### Trick 3: Making Boxes

```python
print("+" + "-" * 20 + "+")
print("|" + " " * 20 + "|")
print("|" + "    Hello World!    " + "|")
print("|" + " " * 20 + "|")
print("+" + "-" * 20 + "+")
```

## Common Mistakes

### Mixing Quote Types
```python
# ❌ Wrong - started with " but ended with '
print("Hello!')

# ✅ Correct - match your quotes!
print("Hello!")
print('Hello!')
```

### Forgetting Quotes Around Text
```python
# ❌ Wrong
print(Hello World)

# ✅ Correct
print("Hello World")
```

✅ Text goes inside quotation marks  
✅ You can use `"double"` or `'single'` quotes  
✅ `*` can repeat text

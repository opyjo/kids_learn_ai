## 📋 Lesson Overview

### Purpose

This lesson reviews if/else and elif statements to keep students' decision-making skills fresh. This is the second coding review lesson in Term 5, maintaining Python skills while exploring AI concepts. Students will practice decision-making with fun new challenges.

### Learning Objectives

By the end of this lesson, students will be able to:

1. Review and use if/else statements confidently
2. Review and use elif statements
3. Combine decisions creatively
4. Solve problems using decision-making
5. See how decisions connect to AI

### Materials Needed

- Computer with internet access (for each student)
- Projector/screen share for demonstrations
- Trinket accounts
- Whiteboard for showing structure

### Pre-Lesson Preparation

1. **Review if/else/elif** — Refresh your understanding
2. **Prepare fun challenges** — Games, helpers, etc.
3. **Think about AI connections** — How decisions power AI
4. **Prepare examples** — Clear, engaging examples

---

### Part 2: If/Else Review (10 minutes)

#### Goals

- Quick review of if/else
- Show examples
- Get students coding

#### The Review

> "Quick review: What does if/else do?"

Wait for: Makes decisions, checks conditions, etc.

**Show examples:**
```python
score = 85
if score >= 80:
    print("You passed!")
else:
    print("Try again!")
```

**Practice:**
> "Create your own if/else program!"

Guide students.

#### Teaching Tips

- Keep it brief
- Get them coding quickly
- Help those who forgot

---

### Part 3: Elif Review (10 minutes)

#### Goals

- Quick review of elif
- Show examples
- Practice

#### The Review

> "What about elif? When do we use it?"

Wait for: Multiple options, etc.

**Show examples:**
```python
if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
# ... etc
```

**Practice:**
> "Create a program using elif!"

Guide students.

#### Teaching Tips

- Keep it brief
- Emphasize order (highest to lowest)
- Get them practicing

## 🎯 Assessment & Differentiation

### Formative Assessment

**During class, observe:**
- [ ] Can student use if/else?
- [ ] Can student use elif?
- [ ] Can student combine decisions?
- [ ] Is student engaged?

### Homework Assessment: "My Decision-Making Program"

| Criteria | Exceeds (3) | Meets (2) | Developing (1) |
|----------|-------------|-----------|----------------|
| **If/else usage** | Uses 5+ times creatively | Uses 3 times | Fewer than 3 |
| **Elif usage** | Uses 3+ times | Uses 1 time | Missing elif |
| **Creativity** | Highly creative | Creative | Basic |
| **Code runs** | No errors | Minor issues | Major errors |
| **Comments** | Clear comments | Some comments | No comments |

### Differentiation Strategies

#### For Students Who Need More Support

- Provide templates
- Focus on if/else first
- Reduce elif requirement
- Pair with a peer
- Provide step-by-step guidance

#### For Advanced Students

- Challenge with complex decisions
- Add and/or combinations
- Encourage creative projects
- Ask them to help peers
- Challenge with nested decisions

## ⚠️ Common Pitfalls & Solutions

### Technical Issues

| Problem | Prevention | Solution |
|---------|------------|----------|
| Forgot elif syntax | Quick review | Show examples |
| Wrong order in elif | Emphasize highest to lowest | "Check biggest first!" |
| Missing colons | Emphasize colons | "Don't forget colons!" |

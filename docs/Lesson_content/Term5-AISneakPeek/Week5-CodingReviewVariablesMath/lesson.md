## 📦 Review: Variables

### What Are Variables?

**Variables** are like labeled boxes that store information!

```python
name = "Alex"
age = 10
score = 95
```

**How they work:**
- `name` stores "Alex"
- `age` stores 10
- `score` stores 95

### Using Variables

```python
# Store information
player_name = "SuperCoder"
level = 5
coins = 100

# Use the information
print(f"Player: {player_name}")
print(f"Level: {level}")
print(f"Coins: {coins}")
```

**Output:**
```
Player: SuperCoder
Level: 5
Coins: 100
```

### Changing Variables

```python
coins = 50
print(f"You have {coins} coins")

# Find treasure!
coins = coins + 100
print(f"After treasure: {coins} coins")

# Buy something
coins = coins - 25
print(f"After buying: {coins} coins")
```

**Output:**
```
You have 50 coins
After treasure: 125 coins
After buying: 100 coins
```

## 🔢 Review: Math Operations

### The Four Basic Operations

**1. Addition (+)**

```python
total = 5 + 3
print(total)  # Output: 8
```

**2. Subtraction (-)**

```python
difference = 10 - 4
print(difference)  # Output: 6
```

**3. Multiplication (*)**

```python
product = 6 * 7
print(product)  # Output: 42
```

**4. Division (/)**

```python
quotient = 20 / 5
print(quotient)  # Output: 4.0
```

### Math with Variables

```python
apples = 5
oranges = 3
total_fruit = apples + oranges
print(f"Total fruit: {total_fruit}")  # Output: Total fruit: 8
```

## 🎮 Fun Challenges

### Challenge 1: Game Score Tracker

```python
# Game Score Tracker
level1_score = 100
level2_score = 150
level3_score = 200

total_score = level1_score + level2_score + level3_score

print("=== GAME SCORES ===")
print(f"Level 1: {level1_score} points")

print(f"Level 3: {level3_score} points")
print(f"Total: {total_score} points")
print(f"Average: {average_score:.1f} points")
```

### Challenge 2: Shopping Calculator

```python
# Shopping Calculator
item1_price = 10
item2_price = 15
item3_price = 8

subtotal = item1_price + item2_price + item3_price

print("=== SHOPPING RECEIPT ===")
print(f"Item 1: ${item1_price}")
print(f"Item 2: ${item2_price}")
print(f"Item 3: ${item3_price}")
print(f"Subtotal: ${subtotal}")
print(f"Tax: ${tax:.2f}")
print(f"Total: ${total:.2f}")
```

# Term 5, Lesson 5: Coding Review - Variables & Math! 💻

**Term:** 5 of 8 | **Week:** 5 of 8

## 📋 Lesson Overview

### Purpose

This lesson reviews variables and math operations to keep students' Python skills fresh during the AI exploration term. While Term 5 focuses on AI concepts, it's important to maintain coding skills. This review lesson reinforces foundational concepts while connecting them to AI applications.

### Learning Objectives

By the end of this lesson, students will be able to:

1. Review and use variables confidently
2. Review and use all four math operations
3. Combine variables and math in programs
4. Solve problems using variables and math
5. See how these skills connect to AI

### Materials Needed

- Computer with internet access (for each student)
- Projector/screen share for demonstrations
- Trinket accounts
- Whiteboard for showing examples
- Optional: Math manipulatives for visual learners

### Pre-Lesson Preparation

1. **Review variables and math** — Refresh your understanding
2. **Prepare fun challenges** — Game trackers, calculators, etc.
3. **Think about AI connections** — How variables/math power AI
4. **Prepare examples** — Clear, engaging examples
5. **Plan differentiation** — Support and challenge activities

## 📚 Detailed Teaching Guide

### Part 1: Welcome & Context (5 minutes)

#### Goals

- Set context for review
- Connect to AI learning
- Get students excited about coding again

#### The Welcome

> "Welcome back! We've been exploring AI concepts, which is awesome! But we can't forget our Python skills! Today, we're going to review variables and math with fun challenges. Plus, remember: variables and math are what POWER AI! So you're learning AI skills while you code!"

#### Teaching Tips

- Emphasize that coding skills are important
- Connect to AI (variables/math power AI)
- Build excitement about coding again

---

### Part 2: Variables Review (10 minutes)

#### Goals

- Quick review of variables
- Show examples
- Get students coding

#### The Review

> "Quick review: What are variables?"

Wait for: Boxes that store information, etc.

**Show examples:**
```python
name = "Alex"
age = 10
score = 95
```

**Practice:**
> "Let's create some variables! Open Trinket and create 3 variables about yourself!"

Guide students through creating variables.

#### Teaching Tips

- Keep it brief—this is review
- Get them coding quickly
- Help those who forgot
- Celebrate when they remember

---

### Part 3: Math Review (10 minutes)

#### Goals

- Quick review of math operations
- Show all four operations
- Practice with examples

#### The Review

> "Quick review: What are the four math operations?"

Wait for: +, -, *, /

**Show examples:**
```python
print(5 + 3)   # Addition
print(10 - 4)  # Subtraction
print(6 * 7)   # Multiplication
print(20 / 5)  # Division
```

**Practice:**
> "Try each operation! Create a program that does all four!"

Guide students through.

#### Teaching Tips

- Keep it brief
- Emphasize * for multiplication
- Show division gives decimals
- Get them practicing

---

### Part 4: Practice Challenges (20 minutes)

#### Goals

- Students practice combining variables and math
- Build confidence
- Solve fun problems

#### The Challenges

**Challenge 1: Game Score Tracker**

Guide students through building a score tracker.

**Challenge 2: Shopping Calculator**

Guide students through building a shopping calculator.

**Circulate and help:**
- Help with syntax
- Encourage creativity
- Celebrate progress
- Help those stuck

#### Teaching Tips

- Build step by step
- Help when needed
- Encourage experimentation
- Keep it fun

---

### Part 5: Creative Projects (10 minutes)

#### Goals

- Students create their own programs
- Apply skills creatively
- Build confidence

#### The Freedom

> "Now create your own program using variables and math! Be creative!"

**Give ideas:**
- Game trackers
- Calculators
- Story programs
- Any creative idea!

**Circulate and support:**
- Help brainstorm
- Guide structure
- Celebrate creativity

---

### Part 6: Wrap-up & Homework (3 minutes)

#### Review Key Points

> "What did we review today?"

Quick review:
1. Variables - store information
2. Math - +, -, *, /
3. Combine them - powerful programs!

#### Introduce Homework

> "For homework, create a program using variables and math!"

#### Preview Next Week

> "Next week, we'll review decision-making!"

## 🎯 Assessment & Differentiation

### Formative Assessment

**During class, observe:**
- [ ] Can student use variables?
- [ ] Can student do math operations?
- [ ] Can student combine them?
- [ ] Is student engaged?

### Homework Assessment: "My Variables & Math Program"

| Criteria | Exceeds (3) | Meets (2) | Developing (1) |
|----------|-------------|-----------|----------------|
| **Variables** | Uses 7+ variables effectively | Uses 5 variables | Fewer than 5 |
| **Math operations** | Uses all 4 creatively | Uses all 4 | Missing operations |
| **Creativity** | Highly creative | Creative | Basic |
| **Code runs** | No errors | Minor issues | Major errors |
| **Comments** | Clear comments | Some comments | No comments |

### Differentiation Strategies

#### For Students Who Need More Support

- Provide templates
- Focus on 2-3 variables first
- Reduce to 2-3 math operations
- Pair with a peer
- Provide step-by-step guidance

#### For Advanced Students

- Challenge with more variables
- Add more complex math
- Encourage creative projects
- Ask them to help peers
- Challenge with real-world problems

---

## ⚠️ Common Pitfalls & Solutions

### Technical Issues

| Problem | Prevention | Solution |
|---------|------------|----------|
| Forgot variable syntax | Quick review | Show examples |
| Using x for multiply | Emphasize * | "Remember: star for multiply!" |
| Confused by division decimals | Explain .0 | "Division gives decimals—normal!" |

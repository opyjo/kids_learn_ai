## 📋 Lesson Overview

### Purpose

This is Part A of the graduation project. Students build a Quiz Game using lists to store questions and answers, then loop through them. This demonstrates mastery of lists, loops, and combining concepts. Students will continue with Part B (AI Presentation) next week.

### Learning Objectives

By the end of this lesson, students will be able to:

1. Create lists of questions and answers
2. Loop through questions using lists
3. Check answers and track score
4. Build a complete, working quiz game
5. Prepare for graduation presentation

### Materials Needed

- Computer with internet access (for each student)
- Projector/screen share for demonstrations
- Trinket accounts
- Whiteboard for project planning
- Optional: Example quiz games

### Pre-Lesson Preparation

1. **Review project requirements** — Know what's required
2. **Prepare example** — Have a working quiz game ready
3. **Plan time management** — Structure the 60 minutes
4. **Prepare troubleshooting** — Common issues students will face
5. **Think about differentiation** — How to support all students

## 📚 Detailed Teaching Guide

### Part 1: Project Introduction (10 minutes)

#### Goals

- Explain project requirements
- Show example quiz game
- Build excitement
- Answer questions

#### Script/Talking Points

> "Welcome to PROJECT WEEK! 🎉 This is Part A of your graduation project. You're going to create a Quiz Game using lists! This combines everything you've learned and prepares you for graduation!"

**Show Example Project:**

Run through a complete example:
1. Show welcome screen
2. Show question and answer lists
3. Demonstrate quiz loop
4. Show score tracking
5. Show final results

> "This is what you're building! But make it YOURS!"

**Explain Requirements:**

Go through checklist:
- Welcome screen ✓
- Question list (5+ questions) ✓
- Answer list (matching order) ✓
- Quiz loop ✓
- Score tracking ✓
- Final results ✓

#### Teaching Tips

- Be enthusiastic
- Show complete example
- Make requirements clear
- Encourage creativity

---

### Part 2: Planning Time (5 minutes)

#### Goals

- Students plan their quiz
- Think about questions
- Prepare to build

#### Activity

> "Before coding, let's plan! Think about:"
- What theme? (Python, games, animals, etc.)
- What questions? (At least 5)
- What answers?

Give students 5 minutes to plan. Walk around and help.

#### Teaching Tips

- Encourage planning
- Help with question ideas
- Keep it brief

---

### Part 3: Building Time (35 minutes)

#### Goals

- Students create quiz games
- Provide support
- Encourage independence

#### Structure

**First 10 minutes:**
- Students start building
- Help with lists setup
- Check in with each student

**Middle 15 minutes:**
- Students build quiz loop
- Circulate and help
- Fix bugs as they arise

**Last 10 minutes:**
- Students finish features
- Add score tracking
- Test their code

#### Teacher Circulation

- Check progress
- Help with bugs
- Encourage
- Differentiate
- Answer questions

#### Common Issues to Watch For

1. **Lists not in sync** — Questions and answers must match
2. **Index errors** — Using wrong indices
3. **Case sensitivity** — Answers not matching due to case
4. **Score not updating** — Update inside loop

#### Teaching Tips

- Stay mobile
- Don't give answers—guide
- Celebrate small wins
- Help students help each other

---

### Part 4: Testing & Polish (7 minutes)

#### Goals

- Students test complete games
- Fix remaining bugs
- Add final touches

#### Activity

> "Time to test! Run through your complete quiz:"
- Does welcome screen show?
- Do questions display?
- Does answer checking work?
- Does score track?
- Are final results shown?

Give students time to test and fix. Help with any remaining problems.

#### Teaching Tips

- Encourage thorough testing
- Help fix final bugs
- Celebrate completed projects

---

### Part 5: Wrap-up & Preview (3 minutes)

#### Goals

- Celebrate achievements
- Preview Part B
- Set expectations

#### Script

> "WOW! Look what you've created! Complete quiz games!"

**Preview Next Week:**

> "Next week is Part B—your AI Presentation! You'll choose an AI topic and create a presentation about it. Then you'll be ready for graduation!"

**Homework Reminder:**

> "For homework, finish polishing your quiz game. Make sure it's perfect!"

#### Teaching Tips

- Celebrate achievements
- Build excitement for Part B
- Set clear expectations

## 🎯 Assessment & Differentiation

### Formative Assessment

**During class, observe:**

- [ ] Can student create question/answer lists?
- [ ] Can student loop through questions?
- [ ] Can student check answers?
- [ ] Can student track score?
- [ ] Is project complete and working?

### Project Assessment: "Quiz Game"

| Criteria              | Exceeds (3)                    | Meets (2)                  | Developing (1)           |
| --------------------- | ------------------------------ | -------------------------- | ------------------------ |
| **Welcome screen**    | Creative, well-formatted       | Basic welcome              | Missing or incomplete    |
| **Question list**      | 10+ questions, creative        | 5+ questions               | Less than 5              |
| **Answer list**        | Matches questions perfectly    | Matches questions          | Doesn't match            |
| **Quiz loop**          | Works perfectly, enhanced      | Works correctly            | Doesn't work             |
| **Score tracking**     | Works perfectly, creative     | Works correctly            | Missing or incorrect     |
| **Final results**      | Creative, well-formatted      | Basic results              | Missing or incomplete    |
| **Code quality**       | Well-commented, organized      | Some comments              | No comments, messy       |
| **Runs without errors** | Perfect, bonus features       | Runs correctly             | Has errors               |

### Differentiation Strategies

#### For Students Who Need More Support

- Provide starter template
- Focus on 5 basic questions first
- Pair with peer helper
- Break into smaller steps
- Reduce to essential features

#### For Advanced Students

- Challenge with 10+ questions
- Add multiple choice
- Add difficulty levels
- Help other students
- Create multiple quiz themes

## ⚠️ Common Pitfalls & Solutions

### Technical Issues

| Problem                  | Prevention                    | Solution                                    |
| ------------------------ | ----------------------------- | ------------------------------------------- |
| Lists not in sync         | Emphasize matching order       | "Question[0] must match answer[0]!"         |
| Case sensitivity         | Show .lower() early           | "Use .lower() to ignore case!"             |
| Index errors             | Show correct indices          | "Use range(len(questions))"                |
| Score not updating       | Show where to update          | "Update score inside the loop!"            |

### Time Management Issues

| Problem                  | Signs                          | Solution                                    |
| ------------------------ | ------------------------------ | ------------------------------------------- |
| Running out of time      | Not finished                   | Prioritize: core features first            |
| Perfectionism            | Stuck on one feature           | "Get it working, then improve"             |

# KidsLearnAI Year 1 Curriculum (Revised)

## 8-Term Python-First Approach for Ages 9-10

_Empowering the Next Generation with AI Education_

---

## Curriculum Philosophy

### Designed for 9-10 Year Olds

This curriculum recognizes that 9-10 year olds learn differently than older kids. They need:

- **More repetition** — concepts need time to sink in
- **Shorter attention spans** — activities change every 10-15 minutes within a session
- **Visual and creative projects** — games, stories, art, humor
- **Concrete thinking** — abstract concepts (like functions with returns) are hard
- **Celebration and encouragement** — badges, showcases, and wins matter

### What We Removed (Saved for Year 2, Ages 11-13)

| Concept                           | Why It's Hard for 9-10          |
| --------------------------------- | ------------------------------- |
| Nested loops                      | Too abstract, hard to visualize |
| Functions with parameters/returns | Requires abstract thinking      |
| Dictionaries                      | Key-value concept is confusing  |
| File I/O                          | Error-prone, frustrating        |
| Pandas/data analysis              | Way too advanced                |
| ML model building                 | Requires all of the above       |

### The AI Promise

Parents enroll their kids for "AI education" — and we deliver by:

1. Teaching the **foundation** AI is built on (Python)
2. Building **computational thinking** skills
3. Introducing **AI concepts** in age-appropriate ways (Term 5 & 8)
4. Preparing them for **real AI projects** in Year 2 (ages 11-13)

## Program Structure

- **Total Duration:** 8 terms (approximately 1 year)
- **Term Length:** 8 weeks per term
- **Session Duration:** 60 minutes live instruction
- **Class Size:** Maximum 10-12 students (smaller for this age)
- **Pacing:** Flexible — students advance when ready

## Term 1: Hello Python!

_"Your first conversation with a computer"_

### What Students Learn

Students discover that they can tell computers what to do using Python. They learn to display messages, store information in variables, and create fun text-based outputs.

### Weekly Breakdown

| Week | Topic                 | What We Do                                                         |
| ---- | --------------------- | ------------------------------------------------------------------ |
| 1    | Welcome to Coding!    | What is programming? Set up our coding space. Type our first code! |
| 2    | Print is Your Voice   | Use print() to make Python talk. Experiment with messages.         |
| 3    | Variables are Boxes   | Store words and numbers in "boxes" (variables). Give them names.   |
| 4    | Playing with Text     | Combine text, make silly sentences, use CAPS and lowercase.        |
| 5    | Practice & Play       | Reinforce concepts with fun challenges and games.                  |
| 6    | Debugging Adventures  | Finding and fixing mistakes (bugs) — it's like being a detective!  |
| 7    | Project: Joke Machine | Build a program that tells jokes, one at a time.                   |
| 8    | Showcase Party!       | Share our Joke Machines, celebrate, get excited for Term 2.        |

### Term 1 Project: Joke Machine

A program that:

- Welcomes the user
- Asks "Want to hear a joke?"
- Tells jokes using print()
- Uses variables to store the jokes

**Example Output:**

```
Welcome to the Joke Machine!
Ready for a joke? Here it comes...

Why don't scientists trust atoms?
Because they make up everything!

Ha ha ha! Want another? Run me again!
```

## Term 2: Math Wizard

_"Turn Python into your personal calculator"_

### What Students Learn

Students discover that Python is amazing at math. They learn to get information from users with input() and create interactive programs that calculate things.

### Weekly Breakdown

| Week | Topic                  | What We Do                                                        |
| ---- | ---------------------- | ----------------------------------------------------------------- |
| 1    | Python Does Math       | Addition, subtraction, multiplication, division with Python.      |
| 2    | Bigger Math            | Exponents, remainder (modulo), order of operations.               |
| 3    | Asking Questions       | Use input() to get information from users.                        |
| 4    | Numbers vs Text        | Converting input to numbers (int). Why it matters.                |
| 5    | Math Challenges        | Solve fun problems: age calculator, tip calculator, etc.          |
| 6    | Practice & Debug       | More practice, finding errors, building confidence.               |
| 7    | Project: My Calculator | Build a personal calculator that asks for numbers and calculates. |
| 8    | Showcase + Badge!      | Present calculators, earn "Python Beginner" badge!                |

### Term 2 Project: Personal Calculator

A program that:

- Asks the user for two numbers
- Adds, subtracts, multiplies, and divides them
- Shows all the results nicely

**Example Output:**

```
Welcome to My Calculator!
Enter your first number: 15
Enter your second number: 4

Here are your results:
15 + 4 = 19
15 - 4 = 11
15 × 4 = 60
15 ÷ 4 = 3.75

Thanks for calculating with me!
```

🏆 **Milestone: "Python Beginner" Badge**

## Term 3: Decision Maker

_"Teach your program to think"_

### What Students Learn

Students learn that programs can make decisions using if and else. This is where programs start feeling "smart" — they respond differently based on what users do.

### Weekly Breakdown

| Week | Topic                   | What We Do                                         |
| ---- | ----------------------- | -------------------------------------------------- |
| 1    | True or False           | Booleans! Things that are either True or False.    |
| 2    | Comparing Things        | Equal to, not equal, greater than, less than.      |
| 3    | If This, Then That      | Write our first if statements.                     |
| 4    | Otherwise... Else!      | Adding else for when the condition isn't true.     |
| 5    | Practice Decisions      | Password checkers, age gates, yes/no games.        |
| 6    | Story Choices           | Make stories that change based on user choices.    |
| 7    | Project: Mini Adventure | Build a short choose-your-own-adventure story.     |
| 8    | Showcase Time!          | Play each other's adventures, share and celebrate. |

### Term 3 Project: Mini Adventure Game

A program that:

- Tells a short story
- Asks the player to make a choice
- Has 2-3 different endings based on choices

**Example Output:**

```
THE MYSTERIOUS DOOR

You're walking through a forest and find a door in a tree.
Do you want to open it? (yes/no): yes

You open the door and find... a tiny dragon!
The dragon offers you a cookie.
Do you take it? (yes/no): yes

The cookie is delicious! The dragon becomes your friend.
THE END - You got the FRIENDSHIP ending!
```

## Term 4: More Choices

_"Handle any situation"_

### What Students Learn

Students expand their decision-making skills with elif (else if) to handle multiple options. They also learn to combine conditions with "and" and "or".

### Weekly Breakdown

| Week | Topic                  | What We Do                                             |
| ---- | ---------------------- | ------------------------------------------------------ |
| 1    | Review If/Else         | Quick review with fun new examples.                    |
| 2    | What About Elif?       | When you have more than two options.                   |
| 3    | Many Options           | Build menus and multi-choice programs.                 |
| 4    | And & Or               | Combining conditions — "if sunny AND warm"             |
| 5    | Grade Calculator       | Build a program that gives letter grades.              |
| 6    | Quiz Builder           | Create a quiz with right/wrong answers and scoring.    |
| 7    | Project: Big Adventure | Expand the adventure game with more choices and paths. |
| 8    | Showcase + Badge!      | Present adventures, earn "Decision Master" badge!      |

### Term 4 Project: Expanded Adventure Game

A program that:

- Has a longer story with more branches
- Uses elif for multiple choices (not just yes/no)
- Has at least 4 different endings
- Includes a simple scoring or inventory system

**Example Output:**

```
THE SPACE EXPLORER

You land on a strange planet. There are three paths:
1. Go to the glowing cave
2. Walk toward the purple forest
3. Investigate the crashed ship

Enter your choice (1, 2, or 3): 2

You enter the purple forest. The trees are singing!
An alien appears and offers you three gifts:
1. A map
2. A laser sword
3. A translator device

Enter your choice (1, 2, or 3): 3

With the translator, you understand the trees' song.
They're giving you directions to hidden treasure!

You found the TREASURE ending! Score: 100 points
```

🏆 **Milestone: "Decision Master" Badge**

## Term 5: AI Sneak Peek

_"Discover the world of Artificial Intelligence"_

### What Students Learn

A breather term! Students step back from heavy coding to explore what AI is, where they encounter it daily, and how everything they've learned connects to AI. Light coding review keeps skills fresh.

### Weekly Breakdown

| Week | Topic                           | What We Do                                                   |
| ---- | ------------------------------- | ------------------------------------------------------------ |
| 1    | What IS AI?                     | Define AI, bust myths (no, robots aren't taking over).       |
| 2    | AI in Your Life                 | Find AI: YouTube recommendations, Siri, video games.         |
| 3    | Smart vs Programmed             | How is AI different from the programs we've made?            |
| 4    | How AI Learns                   | Simple explanation: AI learns from examples (like us!).      |
| 5    | Coding Review: Variables & Math | Fun review challenges using what we learned.                 |
| 6    | Coding Review: Decisions        | More review with new creative twists.                        |
| 7    | Project: AI in My Life Poster   | Create a poster (digital or physical) showing AI around you. |
| 8    | Presentation + Badge!           | Present posters, earn "AI Curious" badge!                    |

### Term 5 Project: AI in My Life Poster

Students create a poster showing:

- 5+ examples of AI they use or encounter
- What each AI does
- Whether they think it's helpful or not
- One question they have about AI

This can be:

- Made in Canva, Google Slides, or PowerPoint
- Hand-drawn and photographed
- A simple Python program that displays AI facts

🏆 **Milestone: "AI Curious" Badge**

## Term 6: Loop Magic

_"Do things over and over (without getting tired)"_

### What Students Learn

Students discover loops — the ability to repeat actions. They use for loops to create patterns, count, and make their programs much more powerful.

### Weekly Breakdown

| Week | Topic                        | What We Do                                                        |
| ---- | ---------------------------- | ----------------------------------------------------------------- |
| 1    | Why Repeat?                  | Understand why loops matter (imagine writing "Hello" 100 times!). |
| 2    | For Loops Intro              | Basic for loop with range(). Count to 10!                         |
| 3    | Loop Challenges              | Count by 2s, 5s, countdown from 10.                               |
| 4    | Patterns with Loops          | Print stars, lines, simple shapes.                                |
| 5    | Creative Patterns            | Make more complex ASCII art patterns.                             |
| 6    | Loops + Decisions            | Combine if statements inside loops.                               |
| 7    | Project: ASCII Art Generator | Create a program that draws cool patterns.                        |
| 8    | Art Gallery Showcase!        | Display our ASCII art, celebrate creativity.                      |

### Term 6 Project: ASCII Art Generator

A program that:

- Uses loops to create visual patterns
- Offers 3+ different patterns to choose from
- Lets user customize (size, character used, etc.)

**Example Output:**

```
Welcome to the Pattern Maker!

Choose a pattern:
1. Triangle
2. Square
3. Diamond
4. Stairs

Enter choice: 1
How big? (1-10): 5

Here's your triangle:

*
**
***
****
*****

Want another pattern? Run me again!
```

## Term 7: Game Builder

_"Create games that keep going"_

### What Students Learn

Students learn while loops — loops that continue until something happens. This unlocks "real" game development: games that don't end until the player wins, loses, or quits.

### Weekly Breakdown

| Week | Topic                  | What We Do                                           |
| ---- | ---------------------- | ---------------------------------------------------- |
| 1    | While Loops Intro      | Loops that continue while something is true.         |
| 2    | Stopping Loops         | How to end a while loop (avoiding infinite loops!).  |
| 3    | Game Loops             | Keep playing until the user says "quit".             |
| 4    | Random Numbers         | Import random, make unpredictable games.             |
| 5    | Keeping Score          | Track points, attempts, wins across rounds.          |
| 6    | Putting It Together    | Combine everything: loops, decisions, random, score. |
| 7    | Project: Guessing Game | Build a full number guessing game.                   |
| 8    | Game Showcase + Badge! | Tournament play, earn "Game Developer" badge!        |

### Term 7 Project: Number Guessing Game

A program that:

- Picks a random number (1-20 or 1-100)
- Lets player guess until they get it
- Gives "higher/lower" hints
- Counts attempts
- Asks if they want to play again

**Example Output:**

```
GUESS MY NUMBER!

I'm thinking of a number between 1 and 20.
Can you guess it?

Your guess: 10
Too low! Try again.

Your guess: 15
Too high! Try again.

Your guess: 12
YOU GOT IT!

You guessed it in 3 tries!

Play again? (yes/no): no

Thanks for playing! Your best score: 3 tries
```

🏆 **Milestone: "Game Developer" Badge**

## Term 8: AI Explorer

_"Combine everything and explore AI deeper"_

### What Students Learn

The grand finale! Students learn lists (their first data structure), build a more complex project, and dive deeper into AI concepts. They create a final project and presentation for graduation.

### Weekly Breakdown

| Week | Topic                         | What We Do                                              |
| ---- | ----------------------------- | ------------------------------------------------------- |
| 1    | Lists Introduction            | Store multiple things in one variable.                  |
| 2    | Working with Lists            | Access items, add items, list length.                   |
| 3    | Loops + Lists                 | Go through every item in a list.                        |
| 4    | AI Deep Dive                  | How AI uses LOTS of data (lists!), pattern recognition. |
| 5    | AI Ethics Chat                | Fairness, privacy, when AI makes mistakes.              |
| 6    | Project Work: Quiz Game       | Build a quiz game using lists of questions/answers.     |
| 7    | Project Work: AI Presentation | Create presentation on an AI topic of choice.           |
| 8    | GRADUATION!                   | Present projects, receive Year 1 Certificate!           |

### Term 8 Projects (Two Parts)

#### Part A: Quiz Game

A program that:

- Stores questions in a list
- Stores answers in a list
- Asks each question, checks answers
- Keeps score
- Shows final results

**Example Output:**

```
PYTHON QUIZ GAME!

Question 1: What do we use to store information?
Your answer: variable
CORRECT!

Question 2: What keyword makes decisions?
Your answer: if
CORRECT!

Question 3: What do we use to repeat code?
Your answer: loop
CORRECT!

FINAL SCORE: 3 out of 3
You're a Python expert!
```

#### Part B: AI Presentation

Students choose ONE topic and create a short presentation:

- How Netflix/YouTube recommends videos
- How voice assistants (Siri/Alexa) work
- AI in video games (NPCs)
- Self-driving cars
- AI art generators
- AI and animals (wildlife tracking, pet apps)

Presentation includes:

- What the AI does
- What data it might use
- Something cool about it
- One concern or question
